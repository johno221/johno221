# Ahoj, som DevOps / Platform Engineer

Zameriavam sa na návrh, automatizáciu a prevádzku moderných infraštruktúr. Moje projekty sú postavené na Kubernetes, Dockeri, CI/CD pipeline-ách a automatizácii pomocou skriptov a DevSecOps nástrojov. Budujem riešenia, ktoré sú spoľahlivé, reprodukovateľné a ľahko udržiavateľné.

## Čomu sa venujem
- Kubernetes (k8s), MicroK8s, Helm, GitOps
- CI/CD systémy: Jenkins, GitLab CI, GitHub Actions
- Docker, kontajnery, build a optimalizácia image-ov
- DevSecOps: Trivy, statická analýza, security scanning
- Monitoring: Prometheus, Grafana, Loki
- Infrastructure as Code: Terraform, Ansible
- Vývoj jednoduchých microservices (Java / Spring Boot, Python)
- Scripting: Bash, Python

## Moje vybrané projekty
- Kafka deployment na Kubernetes pomocou Helm chartov
- PostgreSQL StatefulSet + PVC demo (storage architektúra v k8s)
- Microservice (Spring Boot) + Docker + MicroK8s + JMeter performance testy
- GitOps repo pre osobný web (automatizované nasadenie)
- CI/CD utility pre Jenkins a automatizované POM/JSON upgrady

## Ako pracujem
- Uprednostňujem automatizáciu pred manuálnymi zásahmi.
- Každý projekt má jasnú štruktúru, deployment postup a CI/CD proces.
- Používam GitOps prístup tam, kde je to možné.
- Sledujem bezpečnosť v pipeline už od build fázy.
- Udržiavam čisté, prehľadné repozitáre a technickú dokumentáciu.
